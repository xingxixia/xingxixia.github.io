安装软件

1.ubuntu24.04安装miniconda

进入miniconda官网，点击左边的install。

![cuda13.0官网1](....\images\miniconda install.png)

复制进终端

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
sh Miniconda3-latest-Linux-x86_64.sh

或者也可以下载历史版本，Miniconda archive。

2.ubuntu24.04安装pycharm

直接搜索pycharm linux install，然后去官网下载

下载下来是个压缩包，使用tar -xvf 或图形化解压，并把它放在想安装的位置

进入文件夹，再进入bin文件夹，运行pycharm文件就能进入软件了，但是现在还不够，

再点击工具(tools)，然后选择创建桌面文件（忘了叫什么名字了），安装完毕

3.ubuntu24.04安装pytorch2.9.1

先使用miniconda创建虚拟环境，比如

conda create -n Pytorch_test

然后conda activate Pytorch_test

检查一下cuda是否安装，nvcc -V，这里假设安装的是cuda13.0，注意预览版本的pytorch需要python10或者更高，不然会报错

如果是稳定版，需要python3.9或更高

那么执行

pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu130

或者也可以使用其他版本previous versions of PyTorch，注意版本对应

4.ubuntu24.04安装tensorflow2.18.0

先使用miniconda创建虚拟环境，比如

conda create -n Tensorflow_test

然后conda activate Tensorflow_test

检查一下cuda是否安装，nvcc -V，这里假设安装的是cuda12.5，python 是11

执行 pip install tensorflow[and-cuda]==2.18.0

需要注意的是tensorflow从挺早的版本对windows就没有gpu支持了。

安装完成。
