ubuntu安装单cuda和cudnn

2025/11/6 ubuntu ubuntu cuda

提供两种方案

ubuntu24.04安装单个cuda

这里提供两种方法，一种是安装最新的，一种是安装特定版本

以cuda13.0为例子，具体可以去官网查看

cuDNN Archive和CUDA Toolkit Archive

平时所说，安装cuda深度学习框架，实际上就是为了安装cuda toolkit和cudnn，cuda toolkit就是cuda。

我们需要安装三个东西

cuda driver ----- cuda toolkit ----- cudnn

第一种，添加官方源并直接安装最新版

资料引用：

[[Ubuntu 24.04下CUDA 13.0开发环境搭建和测试]](Ubuntu 24.04下CUDA 13.0开发环境搭建和测试 - ArnoldLu - 博客园)

目前，24.04已经是最新的了，请确保这一点

1.下载基础的编译工具和调试工具 build-essential，dkms，wget，curl，git

sudo apt update
sudo apt upgrade -y
sudo apt install -y build-essential dkms wget curl git

2.卸载旧版本并重启，不然有概率报错

sudo apt remove --purge 'nvidia-*' 'cuda-*' 'libnvidia-*' -y
sudo apt autoremove -y
sudo apt autoclean
sudo reboot

3.添加官方仓库源

wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2404/x86_64/cuda-keyring_1.1-1_all.deb
sudo dpkg -i cuda-keyring_1.1-1_all.deb
sudo apt update

4.安装cuda13.0和驱动

sudo apt install -y cuda

5.配置环境变量

echo 'export PATH=/usr/local/cuda-13.0/bin${PATH:+:${PATH}}' | sudo tee -a /etc/profile.d/cuda.sh
echo 'export LD_LIBRARY_PATH=/usr/local/cuda-13.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}' | sudo tee -a /etc/profile.d/cuda.sh
source /etc/profile.d/cuda.sh

6.重启

sudo reboot

7.验证驱动和cuda

nvidia-smi
nvcc --version

此时cuda就已经安装好了

8.安装cudnn

cudnn从9.12.0开始支持cuda13.x(我猜的，因为从9.12.0开始才出现关于cuda13.x的单独安装)，并且从11.1开始分full和jit版本

这里我们安装cudnn9.16.0,full

wget https://developer.download.nvidia.com/compute/cudnn/9.16.0/local_installers/cudnn-local-repo-ubuntu2404-9.16.0_1.0-1_amd64.deb
sudo dpkg -i cudnn-local-repo-ubuntu2404-9.16.0_1.0-1_amd64.deb
sudo cp /var/cudnn-local-repo-ubuntu2404-9.16.0/cudnn-*-keyring.gpg /usr/share/keyrings/
sudo apt-get update
sudo apt-get -y install cudnn9-cuda-13

之后就安装好啦。

第二种，利用官网手动安装特定版本

1.安装编译软件，卸载驱动程序，和上面是差不多的，不过驱动是可选的，如果版本够的话就不用了

sudo apt update
sudo apt upgrade -y
sudo apt install -y build-essential dkms wget curl git
sudo apt remove --purge 'nvidia-*' 'cuda-*' 'libnvidia-*' -y
sudo apt autoremove -y
sudo apt autoclean
sudo reboot

2.安装指定驱动程序

sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get update
sudo apt install nvidia-driver-580

或者是先搜索

sudo add-apt-repository ppa:graphics-drivers/ppa
ubuntu-drivers devices

然后再安装合适的

sudo apt install nvidia-driver-580

除此之外，还可以图形化点击安装：

点击所有应用，点击附加驱动或是软件与更新(Software & Updates)-其他驱动程序(Additional Drivers )

3.检查驱动

nvidia-detector 或是nvidia-smi

4.安装cuda

来到nvidia的cuda13.0官网CUDA Toolkit 13.0 Downloads

按照次序点击Linux—X86_64—ubuntu—24.04

可以选择runfie形式安装

wget https://developer.download.nvidia.com/compute/cuda/13.0.0/local_installers/cuda_13.0.0_580.65.06_linux.runsudo sh cuda_13.0.0_580.65.06_linux.run

也可以是deb本地安装

wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2404/x86_64/cuda-ubuntu2404.pin
sudo mv cuda-ubuntu2404.pin /etc/apt/preferences.d/cuda-repository-pin-600
wget https://developer.download.nvidia.com/compute/cuda/13.0.0/local_installers/cuda-repo-ubuntu2404-13-0-local_13.0.0-580.65.06-1_amd64.deb
sudo dpkg -i cuda-repo-ubuntu2404-13-0-local_13.0.0-580.65.06-1_amd64.deb
sudo cp /var/cuda-repo-ubuntu2404-13-0-local/cuda-*-keyring.gpg /usr/share/keyrings/
sudo apt-get update
sudo apt-get -y install cuda-toolkit-13-0

又或者是deb网络安装

wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2404/x86_64/cuda-keyring_1.1-1_all.deb
sudo dpkg -i cuda-keyring_1.1-1_all.deb
sudo apt-get update
sudo apt-get -y install cuda-toolkit-13-0

别忘了环境变量

echo 'export PATH=/usr/local/cuda-13.0/bin${PATH:+:${PATH}}' | sudo tee -a /etc/profile.d/cuda.sh
echo 'export LD_LIBRARY_PATH=/usr/local/cuda-13.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}' | sudo tee -a /etc/profile.d/cuda.sh
source /etc/profile.d/cuda.sh

5.安装cudnn

cudnn的安装可以用第一种或是deb网络安装

wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2404/x86_64/cuda-keyring_1.1-1_all.deb
sudo dpkg -i cuda-keyring_1.1-1_all.deb
sudo apt-get update
sudo apt-get -y install cudnn

不过这里还有一种压缩包方式的安装

首先在Index of /compute/cudnn/redist/cudnn/linux-x86_64上查找并下载需要的版本的cudnn

记得要看清楚支持的cuda版本

这里选择cudnn-linux-x86_64-9.16.0.29_cuda13-archive.tar.xz下载，现在是2025年11月16号，可以看到是前几天才发布的

解压文件

tar -xvf cudnn-linux-x86_64-9.16.0.29_cuda13-archive.tar.xz

进入文件夹

cd cudnn-linux-x86_64-9.1.0.70_cuda11-archive/

重点来了，现在复制文件

sudo cp lib/* /usr/local/cuda-13.0/lib64/ 
sudo cp include/* /usr/local/cuda-13.0/include/

注意，文件夹内应有lib和include这两个文件夹，但是也有可能是lib64

如果是，那这里就是

sudo cp lib64/* /usr/local/cuda-13.0/lib64/

并且这两句后面的cuda-13.0，要改到对应的版本，或者干脆去/usr/local/下面也能找到对应的cuda文件夹

我也有看到说是

sudo cp include/cudnn*.h /usr/local/cuda-11.8/include
sudo cp lib/libcudnn* /usr/local/cuda-11.8/lib64

的，不顾我用的是前者，这一种我还没用过

到此就结束了，cuda driver,cuda toolkit 和 cudnn都已经安装。

有报错可以尝试询问chatgpt。
