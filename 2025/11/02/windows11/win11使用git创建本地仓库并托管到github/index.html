使用git创建本地仓库并托管到github

先总结一下：

1.创建git本地仓库

就是说要先有一个本地仓库（就是项目文件夹，不管怎么得到的文件夹总之有就行）

然后cd也好直接open git bush here总之就是在终端页面路径是进入了这个文件夹

执行git init之后这个文件夹就变成了git本地仓库

2.与git关联

配置ssh，这个不多说，可以看win11_git安装与配置那篇文章

在github里创建你的项目，名字最好和刚才的项目名字一致

我没有试过不一致的情况，感兴趣的可以试一试

然后在git bush里执行

git remote add origin 你的github仓库地址

最后执行

git add .添加所有文件

git cmmit -m "消息类型"上传的消息类型

git push -u origin main或者是git push就行了

3.也可以使用clone来下载一个仓库的代码，然后git bash进入其中

执行git init

然后这个文件夹就是本地仓库啦，配置好ssh

剩下的和上面差不多，只是在本地仓库和github仓库的创建顺序上有点差别

git add .

git cmmit -m "your_message"

git push
