win11-git安装与配置

2025/11/2 git

主要是配置用户信息和ssh

1.安装：

直接去官网安装最新版Git for Windows

配置用户信息：

git config --global user.name "your github name"
git config --global user.email "your github email"

检查是否正确：

git config --global user.name

git config --global user.email

用户信息就配置完成了

2.现在配置ssh

先确定自己是否有设置过ssh key

cd ~/.ssh

如果有设置过那就直接cat ~/.ssh/id_rsa.pub

没有的话执行

ssh-keygen -t rsa -C "your github email"

接下来会有三个东西可以设置，不过也可以不设置，嫌麻烦可以直接enter

$ 指定目录: C:\deskbook\（说明：若在此处不输入路径，而直接按回车，则ssh keys生成后存放的路径为C:\User.ssh）

$ 输入密码: 123456

$ 确认密码: 123456

cat ~/.ssh/id_rsa.pub（或者cat你刚才设置的路径，总之打开id_rsa.pub这个文件）

复制里面所有的东西

打开github，点击头像

点击设置，配置ssh，点击new ssh，随便写个名字，然后ssh key就是刚才复制的东西

完事之后再回到git bush

执行

ssh -T git@github.com

看看是否返回

Hi <username>! You've successfully authenticated, but GitHub does not provide shell access

或者诸如此类的话，有就是成功。
