vmware问题和解决办法

2025/11/3 vmware vmware 虚拟机

虚拟机真不行啊，创建是方便了，bug一堆，修半天

1.vmware安装ubuntu20.04后分辨率不能自行调节且无法启用复制粘贴和文件复制功能：

这是vmtools出问题导致的，执行sudo apt-get install open-vm-tools-desktop

2.在虚拟机设置里的网络适配器中打开桥接模式后无法使用网络的问题，但是net模式是正常的：

如果之前有重装过系统，那么这个可能是重装系统导致的

上图是首页，点击下方需要管理员权限才行的<更改设置>

上图是点击后的样子，如果重装过系统或者或者卸载过vmware，可能就不会出现vmnet0（我这里已经修理过了）

点击vmnet1或vmnet8，再点击移除网络，二者都移除后，点击下方的还原默认设置，按照要求来，设置完成后应该就能看到vmnet0了，此时可以用桥接模式了。
